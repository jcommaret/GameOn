# GameOn Project

## Overview

GameOn is a website that allows users to register to a game event.

## Tech Stack

- Vite
- SCSS
- TypeScript
- Yup for validation
- Zustand for state management
- React Hook Form for form handling
- React Router for routing
- FontAwesome for icons

## Installation

```bash
npm install
```

## Usage

```bash
npm run dev
```
